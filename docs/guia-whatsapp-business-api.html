Fase 4: Obter CNPJ

O CNPJ é obrigatório para verificação da empresa no Meta Business e para solicitar a API do WhatsApp Business.

Opções de Abertura

MEI (Microempreendedor Individual)

  • Faturamento até R$ 81.000/ano
  • Abertura gratuita e rápida
  • CNAE sugerido: 6201-5/00 (Desenvolvimento de software)
  • Site: gov.br/mei

ME/EPP (Simples Nacional)

  • Faturamento maior que R$ 81.000/ano
  • Requer contador
  • Mais flexibilidade de CNAEs
  • Processo via Junta Comercial

Documentos que Serão Necessários

  • Cartão CNPJ (emitido pela Receita Federal)
  • Contrato Social ou CCMEI (Certificado MEI)
  • Comprovante de endereço comercial
  • Documento de identidade do responsável

Fase 5: Verificar Empresa no Meta Business

5.1 Acessar Meta Business Suite

Link: https://business.facebook.com/settings/info

5.2 Passos para Verificação

  1. Acesse Configurações de Negócios
  2. Vá em Central de Segurança → Verificação
  3. Clique em "Iniciar verificação"
  4. Preencha os dados da empresa (razão social = nome no CNPJ)
  5. Envie os documentos solicitados:
    • Cartão CNPJ ou Contrato Social
    • Comprovante de endereço (conta de luz/telefone)
  6. Aguarde análise (1-5 dias úteis)

Importante: O nome da Página do Facebook deve corresponder ao nome da empresa no CNPJ para evitar rejeição.

Fase 6: Solicitar WhatsApp Business API

6.1 Criar App no Meta Developers

Link: https://developers.facebook.com/apps/

  1. Clique em "Criar App"
  2. Escolha tipo: "Empresa"
  3. Nome do App: "Horário Inteligente"
  4. Vincule à sua conta do Meta Business verificada
  5. No painel do App, adicione o produto "WhatsApp"
  6. Siga o assistente de configuração

6.2 Configurar Número de Telefone

Atenção: O número usado na API NÃO pode estar vinculado a uma conta pessoal do WhatsApp. Use um número novo ou desvincule o existente antes.

  1. Na seção WhatsApp, clique em "Adicionar número de telefone"
  2. Insira o número com código do país (+55)
  3. Verifique via SMS ou ligação
  4. Configure o nome de exibição (Horário Inteligente)

6.3 Obter Credenciais

Após configuração, você terá acesso a:

Credencial Onde Encontrar
Access Token WhatsApp → Configuração da API → Token de acesso
Phone Number ID WhatsApp → Configuração da API → ID do número
Business Account ID WhatsApp → Configuração da API → ID da conta comercial

Após Obter as Credenciais

Envie as credenciais (Access Token, Phone Number ID e Business Account ID) para configuração no Evolution API. O sistema será atualizado para usar a API oficial do WhatsApp Business.
